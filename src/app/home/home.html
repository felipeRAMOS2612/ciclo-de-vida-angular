Visualizador de Ciclos Personales

Una herramienta interactiva para registrar y visualizar tus eventos personales.

¿Qué es esto?

Una herramienta visual y dinámica para registrar eventos personales y ver cómo se distribuyen dentro de ciclos que se repiten con el tiempo. Utiliza una interfaz moderna basada en Bootstrap y almacenamiento local para mantener tus datos seguros sin depender de servidores.

Características Principales

📅 Registro de eventos

Añade eventos con categoría, fecha y descripción.

🎯 Visualización cíclica

Representación circular de los ciclos en colores.

🎛️ Filtrado por categoría

Selecciona el tipo de ciclo para visualizar solo eventos relevantes.
